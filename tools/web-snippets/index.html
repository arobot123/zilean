前端快速查询

本页面列出了个人对WEB的理解。并学习总结列出常用操作代码块示例，便于开发查询。

Web展示的是什么？我个人觉得Web展示的其实都是在线的文档、流数据等。 Web页面就是一张白纸，每个人都可以按照自己的想法和创意，根据世界约定的规则去展示自己的想法和需要表现的内容（HTML）。 至于展现的是否美观、简洁、丰富等，需要用到绘画工具——CSS。 如果不满足于静态文档，要想页面能和自己发生互动、和互联网上其他资源发生互动，就需要用JavaScript这些工具了。

前端开发其实就是创作的过程，我们展示自己所想，我们遵从相同的规则，世界运转有序。

PS: 为了使工作变得轻松，尽量让各部分做好自己的本职工作，不要让他们相互纠缠，这样会很头痛。。。
